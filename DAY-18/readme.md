# Student Placement Predictor

## Overview

A Machine Learning web application that predicts whether a student is likely to be placed based on academic and skill-related features.

## Features

- Logistic Regression Model
- Streamlit Web App
- Placement Prediction
- Placement Probability
- Model Accuracy Display

## Tech Stack

- Python
- Pandas
- Scikit-Learn
- Streamlit

## Run

```bash
streamlit run app.py
```
